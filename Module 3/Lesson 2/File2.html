1. Що таке HTML?

HTML (HyperText Markup Language) — це мова розмітки, а не мова програмування. Її мета — не виконувати логіку, а надавати структуру та сенс вмісту веб-сторінки. HTML є скелетом, який утримує всі елементи на сторінці.

Коли ваш браузер отримує HTML-документ від сервера, він "читає" розмітку і відображає її відповідно до визначених правил.

2. Основні поняття HTML

Елемент: Це базовий будівельний блок HTML. Він складається з:

  • Тегу, що відкриває (<p>): Початок елемента.
  • Тегу, що закриває (</p>): Кінець елемента.
  • Вмісту: Інформація всередині тегів.
  • Атрибутів: Додаткова інформація про елемент.

Приклад:

<a href="https://example.com">Цей текст - посилання</a>

Цей текст - посилання

Вкладеність (Nesting): Елементи можуть міститися один в одному.

<div>
  <h1>Заголовок</h1>
  <p>Це текст в абзаці.</p>
</div>

3. Структура HTML-документа (Бойлерплейт)

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Назва моєї сторінки</title>
</head>
<body>

  <h1>Привіт, світ!</h1>
  <p>Це мій перший HTML-документ.</p>

</body>
</html>
  • <!DOCTYPE html>: Вказує, що це HTML5 документ.
  • <html>: Кореневий елемент сторінки.
  • <head>: Містить метадані (назву, кодування, стилі).
  • <title>: Назва сторінки у вкладці браузера.
  • <body>: Основний вміст сторінки.

4. Найпоширеніші HTML-теги

Заголовки:

<h1>Найбільший заголовок</h1>
<h6>Найменший заголовок</h6>

H1 Заголовок

H6 Заголовок

Параграфи:

<p>Це звичайний текст абзацу.</p>

Це звичайний текст абзацу.

Посилання:

<a href="https://example.com">Перейти</a>
Перейти

Зображення:

<img src="https://via.placeholder.com/150" alt="Демо-зображення">

Списки:

<ul>
  <li>Елемент 1</li>
  <li>Елемент 2</li>
</ul>
<ol>
  <li>Перший</li>
  <li>Другий</li>
</ol>
  • Елемент 1
  • Елемент 2
  1. Перший
  2. Другий

<span>Це рядковий контейнер</span>
